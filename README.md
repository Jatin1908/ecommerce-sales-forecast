# 🛒 E-commerce Sales Forecasting

This project predicts future sales using time-series analysis.

# 📊 Features
- Sales data visualization
- Trend analysis
- Future sales prediction using ARIMA

# 🧠 Technologies Used
- Python
- Pandas
- Matplotlib
- Statsmodels (ARIMA)

# 📁 Files
- data.csv → dataset
- main.py → main code
